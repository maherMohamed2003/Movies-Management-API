# 🎬 Movie Management API

A RESTful API built with **ASP.NET Core Web API** for managing movies and genres.

## 🚀 Features
- CRUD operations for Movies and Genres
- Repository Pattern
- Entity Framework Core (Code First)
- DTOs for Insert & Update
- Swagger API Documentation
- Tested using Postman
- Deployed to MonsterAPI

## 🛠️ Technologies Used
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Swagger (OpenAPI)
- Postman

## 📂 Project Structure
- Controllers
- Services
- Repositories
- DTOs
- Models
- Data (DbContext & Migrations)
